# SNV4 Font Builder v1.2

A local web-based tool for converting sound fonts into SNV4 format.

## Supports
- Proffie
- Xenopixel
- SNV4-style packs

## Main Features
- ZIP upload
- Automatic root detection
- Audio preview before and after build
- Smart matching and merge decisions
- Advanced edit mode
- Drag-and-drop reordering
- Build to SNV4-ready output
- Download built ZIP or open output folder
- Reset temporary job folders
- Dark / Light theme toggle

## Requirements
- PHP: `php-8.5.4-nts-Win32-vs17-x64`
- Visual C++ Redistributable (VS17)

See `GETTING_PHP.md` for exact setup.

## Output
Built fonts are saved to the main `/output` folder.
