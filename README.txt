SNV4 FONT BUILDER v1.2

Quick Setup

1. Download PHP (see docs/GETTING_PHP.md)
2. Extract the FULL PHP ZIP into the "php" folder
3. Install Visual C++ if needed
4. Run "start-web.bat"

Basic Use

1. Upload your sound font ZIP
2. Select the detected root
3. Review sounds if needed
4. Click Build
5. Output will be in the "output" folder

Supports
- Proffie
- Xenopixel
- SNV4
